# ImpactThone-2026

## AI-Powered Multilingual Grievance Routing System

An AI-driven platform that enables citizens to submit grievances using voice in their native languages and automatically routes complaints to the appropriate government department.

---

## Problem Statement

Existing grievance portals often struggle with regional languages and dialects, causing delays and forcing citizens to depend on intermediaries. This particularly affects rural and non-English-speaking communities.

---

## Proposed Solution

ImpactThone provides a voice-first multilingual grievance system that:

* Accepts complaints in local languages.
* Converts speech into text using AI.
* Translates regional languages into English.
* Detects urgency and prioritizes critical cases.
* Automatically routes complaints to the correct department.
* Improves accessibility and response time.

---

## Features

* 🎤 Voice-to-Text using OpenAI Whisper
* 🌐 Multilingual Support
* 🔄 Automatic Translation to English
* 🧠 AI-based Complaint Classification
